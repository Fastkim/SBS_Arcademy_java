class :     1. 이름은 항상 첫 번째 자는 대문자로 사용한다.
	 2. 클래스는 항상 객체가 생성(만들어야 됨)되어야 한다.
	    (이유는 : 클래스는 비어있는 방 이다. 그래서 값을 저장할 수 없다)
	3. 생성자 : 객체를 초기화 시키는 기능을 가진다.
	    (클래스 이름과 동일하다) ==> 초기화를 보장하기 위해서
	4. 생성자는 클래스가 선언 되면 생성자도 자동 생성
	    


	* 생성자 : 메서드(실행을 하면 반드시 결과값을 반환)
	           : 생성자(결과값을 반환하지 않는다. )

	객체를 만드는 방법
	1. 클래스이름 객체이름 = new 생성자();
	      ==> new : 객체를 만들어주는 연산자
	      ==> 생성자 : 만들어진 객체를 초기화 해주는 메서드
	             (클래스의 이름과 동일)
	   InitSample aa = new InitSample();	

            함수와 메서드의 차이

	함수 : 함수이름() 이렇게 선언하여 사용
	메서드() : 객체를 통해 호출하여야만 실행 가능하다.
		객체.메서드()

	초기화 순서(호출)
	main 클래스 찾기 ==> 0.static ==> 1. 객체 ==> 2. 생성자 ==> 3. 객체.메서드()
	
	생성자 ==> 메서드
	
	(함수)
	메서드 : 반복적으로 코딩하여 일을 수행할 때 한번 만 코딩하여 반복해서 사용, 객체.메서드() 실행
